Introduction

User handler api endpoint documentation

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Admin Auth endpoint

POST Login

Login a user.

Example request:
curl --request POST \
    "http://localhost/api/admins/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"alyson74@example.org\",
    \"password\": \"1>dl#9&|L*Eh\"
}"

const url = new URL(
    "http://localhost/api/admins/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "alyson74@example.org",
    "password": "1>dl#9&|L*Eh"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "access_token": "token"
}

Example response (422):


{
    "email": [
        "The provided credentials are incorrect."
    ]
}

You can check the Dev Tools console for debugging information.

User endpoint

POST Signup

Create a User.

Example request:
curl --request POST \
    "http://localhost/api/users" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"vhdbfpgk\",
    \"email\": \"connelly.ardella@example.org\",
    \"password\": \"3V9=Fd%p*SY\",
    \"division\": \"hnqazoacabijuggkqf\"
}"

const url = new URL(
    "http://localhost/api/users"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "vhdbfpgk",
    "email": "connelly.ardella@example.org",
    "password": "3V9=Fd%p*SY",
    "division": "hnqazoacabijuggkqf"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "data": {
        "id": 1,
        "name": "Random Name",
        "email": "example@example.com",
        "division": "A"
    }
}

Example response (422):

}

division   string   

Must not be greater than 255 characters. Example: hnqazoacabijuggkqf
